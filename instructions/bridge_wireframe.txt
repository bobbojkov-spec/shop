BRIDGE513 – HOME PAGE WIREFRAME (TEXT-ONLY)
==========================================

Viewport assumptions:
- Desktop: 1440px width
- Tablet: 768–1024px width
- Mobile: 360–480px width


[HEADER – fixed at top]
------------------------------------------------------------
| LOGO (left)   | NAV (center)                | ICONS (right) |
|               | Home | About | Stores ...   | [Search] [Cart]|
------------------------------------------------------------

- Desktop:
  - Single horizontal bar
  - Logo left-aligned
  - Menu centered or left-to-center
  - Search + Cart icons on the right
- Mobile:
  - Top row: Logo (left) + Hamburger (right)
  - Full-screen slide-down menu when hamburger tapped


[HERO FEATURE STRIP]
============================================================
| [Card 1] | [Card 2] | [Card 3]                            |
|          |          |                                     |
------------------------------------------------------------
Card structure (each):
  - Top: Image (ceramic/scene)
  - Middle: Heading (e.g., "INSPIRED BY THE SEA")
  - Bottom: short blurb + “learn more” text link/button

Responsive:
- Desktop: 3 cards side-by-side, equal width
- Tablet: likely 2 cards per row
- Mobile: single column, cards stacked with vertical spacing


[PRODUCT GRID SECTION – FEATURED/SHOP PREVIEW]
============================================================
Section layout:
------------------------------------------------------------
|           [Product Grid Title / Intro]                    |
------------------------------------------------------------
| [Product 1] [Product 2] [Product 3] [Product 4]          |
| [Product 5] [Product 6] ...                              |
------------------------------------------------------------
|                      [VIEW ALL]                          |
------------------------------------------------------------

Product card structure:
- Top: Product image (square or portrait)
- Top-left or top-right overlay: wishlist heart icon on hover
- Below image:
  - Product title (e.g., "Rose Oval Plate")
  - Price (e.g., "$300.00")
  - Add to cart button in text or bordered style

Responsive:
- Desktop: 3–4 items per row, equal width
- Tablet: 2–3 per row
- Mobile: 1–2 per row
